The Variation Swatches Master Lite for WooCommerce is an advanced module for WP-powered online stores. It helps you to customize the variations on your product pages in a few clicks to display them into outstanding swatches.

== Description ==

Enhance the visual aesthetics of your products and the attributes of a variation to bestow an appealing look to your product page. It is a feature-packed extension to get a neat, polished, and sophisticated look for all variations on your WooCommerce store pages. 

This user-friendly plugin allows you to select different attributes and styles to display the variations of all variable products. There are endless options and flexibility to customize the look and feel you want to achieve for the swatches- round or square shape, convert dropdown menus into images, and so much more.

Select a preferred style for the variations from several options grouped under different categories- color, images, buttons, and radio. It also allows the buyers on your website to add products to the shopping cart from the store gallery/Archive pages directly using the ‘Add to Cart’ button.

== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install of WooCommerce Order Coupon, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "Variation Swatches Master Lite for WooCommerce" and click Search Plugins. Once you've found our Variation Swatches Master Lite for WooCommerce you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking "Install Now".

= Manual installation =

The manual installation method involves downloading our Variation Swatches Master Lite for WooCommerce and uploading it to your web server via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
